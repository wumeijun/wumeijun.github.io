模拟逐字输入文字的效果

iOS7以后NSString类增加了2个绘制字符串对象的方法

  • drawAtPoint:withAttributes:
  • drawInRect:withAttributes:

使用定时器加字符串的绘制方法模拟写字的过程

  • 定义一个定时器和初始化数据

    self.title = @”我是一条鱼啊啊啊啊，可爱的鱼啊，我是一条鱼啊啊啊啊，可爱的鱼啊,我是一条鱼啊啊啊啊，可爱的鱼啊，我是一条鱼啊啊啊啊，可爱的鱼啊”;

    self.timer = [NSTimer scheduledTimerWithTimeInterval:0.1 target:self selector:@selector(updateCurrentString) userInfo:nil repeats:YES];

  • 每次取出要显示的字符串，依次递增

  • (void)updateCurrentString
    {
    static NSInteger m = 1;
    if (m <= self.title.length) {

    self.currentString = [self.title substringWithRange:NSMakeRange(0,m)];
m++;
//调用drawRect:(CGRect)rect方法进行绘制
[self setNeedsDisplay];

    }else {

    [self.timer invalidate];
self.timer = nil;

    }
    }

  • 每调用之前先清除画布，否则会出现抖动
    第一个显示第一个字，第二次显示头2字，视觉上造成是逐字显示一样

  • (void)drawRect:(CGRect)rect {

    //获取上下文
    CGContextRef ctxxx = UIGraphicsGetCurrentContext();
    //清除画布
    CGContextClearRect(ctxxx, rect);

    UIFont font = [UIFont systemFontOfSize:17];
    UIColor     color = [UIColor redColor];
    NSDictionary *dic = @{

    NSForegroundColorAttributeName:color,
NSFontAttributeName:font
};

    [self.currentString drawInRect:CGRectMake(10, 10, 110, 190) withAttributes:dic];

}
