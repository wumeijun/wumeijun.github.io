CAAnimation 常用的keyPath整理

CAAnimation 常用的keyPath

[CAAnimation animationWithKeyPath: @”key”]

  • formValue和toValue 都是float类型

    transform.scale 缩放
    transform.scale.x 宽的缩放
    transform.scale.y 高的缩放
    transform.rotation.z 类似绕着UIView的中心旋转，平面图的旋转
    opacity 透明度变换

    strokeStart和strokeEnd 路径的开始和结尾

    1 keyPath = strokeStart  fromValue = 0，toValue = 1
反方向路径消失

2 keyPath = strokeStart  动画的fromValue = 1，toValue = 0
正方向画出路径

3 keyPath = strokeEnd  fromValue = 0，toValue = 1
正方向画出路径

4 keyPath = strokeEnd  动画的fromValue = 1，toValue = 0
反方向路径消失
  • position 位置移动 formValue和toValue 是NSValue
  • shadowColor 阴影颜色变化

动画结束后回到初始状态解决方法

animation.removedOnCompletion = NO;
animation.fillMode = kCAFillModeForwards;
